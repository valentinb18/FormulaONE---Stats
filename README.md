# 🚀 FormulaONE---Stats

**Materia:** Programación III — Frontend
**Carrera:** Tecnicatura Universitaria en Programación

## 📋 Descripción

FormulaONE Stats es un sitio web sobre la Fórmula 1 desarrollado como trabajo práctico integrador. Permite consultar información actualizada sobre pilotos, equipos y circuitos de la temporada actual, además de un registro de pilotos históricos y un formulario de contacto con validación.

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS
- JavaScript
- Bootstrap 5.3
- Apache ECharts
- Font Awesome
- Google Fonts (Titillium Web + Barlow Condensed)
- API Jolpica F1
- API OpenF1

## 📄 Páginas

- **🏆Inicio🏆** — Presentación del sitio, estadísticas rápidas y pilotos destacados.
- **🏆Pilotos🏆** — Listado de pilotos con filtro en tiempo real y gráficos estadísticos.
- **🏆Equipos🏆** — Constructores de la temporada actual con filtro.
- **🏆Circuitos🏆** — Calendario de carreras 2026 con filtro por país.
- **🏆Históricos🏆** — Los 10 pilotos más ganadores de la historia con filtro.
- **🏆Contacto🏆** — Formulario de contacto con validación en JavaScript.

## 🌐 APIs utilizadas

- **🏎️Jolpica F1🏎️** — Datos de pilotos, equipos y circuitos.
- **🏎️OpenF1🏎️** — Fotos de los pilotos de la grilla actual.

## ⚡ Instrucciones para ejecutar localmente

1. Descargar el archivo .zip y descomprimirlo
2. Abrir la carpeta del proyecto
3. Abrir el archivo `index.html` en el navegador
4. No requiere instalación ni servidor adicional
